# Безпека в хмарі

## Основні загрози
- Несанкціонований доступ.  
- Витоки даних.  
- Атаки DDoS.  
- Помилки конфігурації.

## Методи захисту
- Шифрування даних.  
- Управління ідентичностями (IAM).  
- Багаторівнева аутентифікація.  
- Фаєрволи, IDS/IPS.

## Відповідальність
- Модель Shared Responsibility: провайдер відповідає за інфраструктуру, клієнт — за дані.
